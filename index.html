Vinícola Agnello

Descrição do Projeto: Sistema de Gerenciamento de Vinho O Sistema de Gerenciamento de Vinho é uma aplicação web desenvolvida em JavaScript, projetada para ajudar a vinheria Agnello a organizar e gerenciar seu inventário de vinhos de forma eficiente. O sistema permite que o usuário adicione, edite, visualize e exclua informações sobre diferentes vinhos, mantendo um registro detalhado e atualizado de sua coleção. Funcionalidades Principais: Cadastro de Vinho: Permite ao usuário adicionar novos vinhos ao sistema, incluindo informações como nome, tipo (tinto, branco, rosé), safra, preço e origem. Edição de Vinho: O usuário pode atualizar os dados dos vinhos já cadastrados. Exclusão de Vinho: O sistema oferece a opção de excluir vinhos da coleção. Visualização de Inventário: Exibe a lista de vinhos com as informações relevantes, permitindo que o usuário filtre e pesquise por critérios como tipo ou safra. Armazenamento Local: Utiliza o LocalStorage do navegador para armazenar os dados localmente, permitindo que a informação persista mesmo após o fechamento da página. Validação de Dados: O sistema valida as entradas dos usuários para garantir que os dados inseridos sejam válidos e completos.
